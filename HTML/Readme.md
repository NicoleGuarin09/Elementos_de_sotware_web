b. IP 

La IP (Dirección de Protocolo de Internet) es un número único que identifica a cada dispositivo en internet.

c. Protocolo 

Un protocolo es un conjunto de reglas que permiten que los dispositivos se comuniquen entre sí.

d. Nombre de dominio 

Es el nombre fácil de recordar de un sitio web.

e. Directorio y nombre de archivo 

Es la forma en que se organiza la información dentro de un sitio web.

2. Mencionamos dos roles importantes en el desarrollo de un producto web, pero hay 
otros roles involucrados, consulte y defina qué hace cada uno de los siguientes 
roles: 

a. Diseñador UI/UX 

UI (User Interface): diseña cómo se ve la página (colores, botones, diseño).
UX (User Experience): se enfoca en que la experiencia del usuario sea fácil y agradable.

b. Cloud Manager / Ingeniero de infraestructura en la nube 

Se encarga de:

Configurar servidores en la nube (como AWS o Google Cloud)
Mantener la página funcionando
Garantizar seguridad y rendimiento

c. Administrador de base de datos

Se encarga de:

Guardar y organizar la información
Proteger los datos
Optimizar consultas

d. Project Manager / Director del Proyecto 

Se encarga de:

Organizar el equipo
Planear tiempos y tareas
Asegurar que el proyecto se termine bien

3. ¿Qué significa que un lenguaje sea turing completo? 

Un lenguaje es Turing completo cuando puede resolver cualquier problema computacional si tiene suficiente tiempo y memoria.

Es decir: puede hacer cualquier tipo de programa (como Python o Java).
